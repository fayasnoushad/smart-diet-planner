## Frontend

This directory contains the frontend code for the Smart Diet Planner project. It is built using Next.js, TypeScript, and Tailwind CSS, providing a responsive and user-friendly interface.

This is a hackathon project and is not intended for production use.

---

### Setup Instructions

```bash
cd frontend
npm install
# add environment variables in .env file
npm run dev
```

---

### Environment Variables

Create a `.env` file in the `frontend` directory and add the following variables:

```
NEXT_PUBLIC_API_URL="http://localhost:8000"
```
